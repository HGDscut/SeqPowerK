# SeqPowerK
SeqPowerK是一个模拟软件，是为了论文"The statistical power of k-mer based sequence statistics for alignment-free comparison of metagenome contigs"，用于模拟计算基于k-mer的序列非比对方法的统计功效。
该软件只支持在windows平台上使用。
